# Slime Arena v0.3.0 Known Issues

## Интерфейс

### Перед боем

- Не показывать уровень слайма в левом верхнем углу.

### Во время боя

- В начале игры пишется "вы погибли" для тех игроков, которые играют повторно в уже следующую арену. Это некорректно, они только что вошли в новый бой.
- Панель с уровнем и прогрессом до следуюущего перенести из правого верхнего угла в левый верхний угол и объединить с информационной панелью.
- Из инфо-панели удалить дублирующие массу и уровень.
- Макс просто удалить 
- Проверить видимость границ. Верхняя граница сейчас скрыта под панелью браузера (проверено в Safari iOS)

### После боя

- Панель с выбором класса лишняя
- Левая кнопка с таймером не должна сама перекидывать на экран "Перед боем"
- По истечении таймера она должна называться "В бой" и сразу отправлять игрока на новую арену под тем же именем и с тем же классом.
