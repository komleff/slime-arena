# Project Brief
Краткое описание сути проекта.

## Что это?
Slime Arena — casual multiplayer browser-based arena game. Игроки управляют милыми слаймами, собирают массу и сражаются в коротких матчах (2–3 минуты). Платформа: мобильный браузер (основная), браузер на ПК.

## Основные цели
- Создать быструю соревновательную аркаду для браузера
- Обеспечить детерминированную серверную симуляцию
- Реализовать централизованный механизм настройки баланса без правок кода
- Чёткое разделение: клиент — ввод и отображение, сервер — правила и состояние

## Ключевые особенности
- **Управление одним джойстиком** (левая рука) + умения (правая рука)
- **Пасть наносит урон**, хвост уязвим
- **Физика с инерцией и заносом** — дрифт-развороты для опытных
- **Логарифмическое масштабирование** — рост силы, но падение манёвренности
- **Система «Мятежник»** — награда за убийство лидера
- **Hot Zones** вместо классических shrinking circles
- **Pay-to-customize**, не pay-to-win
