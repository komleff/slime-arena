# Project Brief
Краткое описание сути проекта.

**Версия GDD:** 3.3  
**План:** SlimeArena-Plan-v3.3.md

## Что это?
Slime Arena — casual multiplayer browser-based arena game. Игроки управляют милыми слаймами, собирают массу и сражаются в коротких матчах (3 минуты). Платформа: мобильный браузер (основная), браузер на ПК.

## Основные цели
- Создать быструю соревновательную аркаду для браузера
- Обеспечить детерминированную серверную симуляцию (30 Гц)
- Реализовать централизованный механизм настройки баланса без правок кода
- Чёткое разделение: клиент — ввод и отображение, сервер — правила и состояние

## Ключевые особенности
- **Управление одним джойстиком** (левая рука) + умения (правая рука)
- **Пасть наносит урон** (120°), хвост уязвим (×1.5 урона)
- **Физика с инерцией и заносом** — дрифт-развороты для опытных
- **Логарифмическое масштабирование** — рост силы, но падение манёвренности
- **Система «Король»** — виден на мини-карте, золотые пузыри при укусе
- **Сундуки** — спорная механика с обручами, таланты и пузыри
- **Pay-to-customize**, не pay-to-win

## Фазы матча (GDD-Core.md)
| Фаза | Время | Особенности |
|------|------|-------------|
| Рост | 0–60 сек | Максимальная плотность ресурсов |
| Охота | 60–120 сек | Ресурсов меньше, чаще конфликты |
| Финал | 120–180 сек | Безопасные зоны, повышенная награда за риск |

## Классы (GDD-Core.md)
| Класс | Пассив | Стартовое умение |
|-------|--------|------------------|
| Охотник | +15% скорость | Рывок |
| Воин | +10% урон укусом слаймов | Щит |
| Собиратель | +25% размер укуса пузырей | Замедление |
