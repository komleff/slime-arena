# Active Context
Текущее состояние проекта и фокус работы.

## Текущее состояние
Проект находится в стадии отладки плавности движения и точности управления. Завершена оптимизация буфера снапшотов с учётом сетевого jitter.

## Последние изменения
- **Шаг 1**: Добавлена диагностическая панель (F3) и расширенное логирование сервера.
- **Шаг 2**: Оптимизация буфера снапшотов:
  - Добавлен `jitterBufferMs` — дополнительный буфер на основе jitter (коэффициент 2x)
  - Асимметричное сглаживание: быстрое увеличение задержки, медленное уменьшение
  - Улучшенный `updateRenderClock` с мягким ограничением вместо резких скачков
  - Метрика `bufferHealth` для оценки здоровья буфера

## Ближайшие шаги
- Шаг 3: Аудит интерполяции углов
- Шаг 4: Тюнинг FBW-системы на сервере
- Шаг 5: Аудит overspeed damping и границ мира
