# UI Extension Index

Обзор пользовательского интерфейса и страниц проекта Slime Arena.

## Страницы
1.  [Game Page](pages/game_page.md) — Основная игровая арена.

## Общие компоненты
- **Canvas Renderer**: Система отрисовки на базе HTML5 Canvas.
- **UI Modals**: Система всплывающих окон (Talents, Results) на базе DOM-элементов поверх Canvas.
- **Input Handlers**: Обработчики клавиатуры и виртуального джойстика.
