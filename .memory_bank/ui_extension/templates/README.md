# UI Templates

В данном проекте не используются классические серверные шаблоны (например, Bitrix или Blade).

## Особенности реализации
- **SPA (Single Page Application)**: Весь интерфейс строится динамически на стороне клиента.
- **Dynamic DOM**: Элементы UI (HUD, модальные окна) создаются через `document.createElement` в `client/src/main.ts`.
- **Canvas**: Игровая сцена отрисовывается процедурно, без использования HTML-шаблонов.

## Найденные структуры
- `client/index.html`: Базовый скелет страницы, подключающий JS-модуль.
