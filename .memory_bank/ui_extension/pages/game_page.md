# Game Page (Main)

Основная и единственная игровая страница проекта.

## Назначение
Предоставление игрового процесса: управление слаймом, взаимодействие с миром и другими игроками.

## Компоненты и виджеты
- **Game Canvas**: Основная область отрисовки игрового мира.
- **HUD (Heads-Up Display)**: Отображение текущей массы, здоровья и времени матча.
- **Virtual Joystick**: Элемент управления для мобильных устройств и тач-скринов.
- **Talent Modal**: Окно выбора талантов при повышении уровня.
- **Results Modal**: Экран итогов матча с таблицей лидеров.

## Входы и выходы данных
- **Входы**:
    - Состояние игры (`GameState`) от сервера через WebSocket.
    - Конфигурация баланса (`BalanceConfig`) при подключении.
- **Выходы**:
    - Команды ввода (`InputCommand`): движение, использование способностей, выбор талантов.

## Реализация
- **HTML**: `client/index.html`.
- **Логика**: `client/src/main.ts`.
- **Рендеринг**: `client/src/rendering/`.
- **Ввод**: `client/src/input/`.
