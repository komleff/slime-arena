# Results Modal Component

Экран завершения матча.

## Назначение
Отображение победителя, итоговой таблицы лидеров и таймера до следующего матча.

## Точки конфигурации
- `resultsDurationSec`: Длительность отображения экрана.

## Параметры
- `winnerName`: Имя победителя.
- `leaderboard`: Список топ-игроков.
- `countdown`: Время до рестарта.

## Зависимости
- DOM API.

## Где вызывается
- `client/src/main.ts`: Обновление при переходе комнаты в фазу `Results`.
