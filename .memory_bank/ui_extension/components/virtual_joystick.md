# Virtual Joystick Component

Элемент управления движением.

## Назначение
Преобразование тач-событий или движений мыши в вектор направления движения слайма.

## Точки конфигурации
- `joystickRadius`: Радиус основания.
- `joystickDeadzone`: Мертвая зона.
- `joystickMode`: Режим (fixed/adaptive).

## Параметры
- `moveX`, `moveY`: Результирующий вектор (от -1 до 1).

## Зависимости
- `client/src/input/joystick.ts`: Логика обработки ввода.

## Где вызывается
- `client/src/main.ts`: Инициализация при старте, обновление в обработчиках событий Pointer.
