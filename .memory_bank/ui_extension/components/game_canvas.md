# Game Canvas Component

Основной компонент для визуализации игрового мира.

## Назначение
Отрисовка всех игровых сущностей (слаймы, орбы, сундуки, зоны) с использованием HTML5 Canvas 2D API.

## Точки конфигурации
- `canvas.width`, `canvas.height`: Размеры холста, обновляются при ресайзе окна.
- `camera`: Объект, определяющий смещение и масштаб отрисовки.

## Параметры
- `renderState`: Текущее интерполированное состояние мира.
- `balanceConfig`: Параметры мира (размеры карты, радиусы объектов).

## Зависимости
- `client/src/rendering/draw.ts`: Утилиты для рисования примитивов и спрайтов.

## Где вызывается
- `client/src/main.ts`: В основном цикле `requestAnimationFrame`.
