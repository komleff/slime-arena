# HUD Component

Информационная панель игрока.

## Назначение
Отображение текущих показателей игрока: масса, здоровье, время до конца матча, количество доступных талантов.

## Точки конфигурации
- Стилизация через CSS в `main.ts`.

## Параметры
- `player.mass`: Текущая масса.
- `player.hp` / `player.maxHp`: Здоровье.
- `state.timeRemaining`: Оставшееся время.
- `player.talentsAvailable`: Количество очков талантов.

## Зависимости
- DOM API.

## Где вызывается
- `client/src/main.ts`: Функция `updateHud()`.
