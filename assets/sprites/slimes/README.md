# Slime Sprites

Папка для спрайтов и скинов слаймов для игроков в Slime Arena.

## Структура

```
slimes/
├── base/           # Базовые спрайты всех классов (обязательные)
├── warrior/        # Спец-спрайты для Warrior класса
├── collector/      # Спец-спрайты для Collector класса
└── effects/        # Эффекты (опционально)
```

## Папка `base/`

Базовые спрайты используются для всех игроков независимо от класса.

**Файлы:**
- `slime-idle.png` - Базовый спрайт слайма в спокойном состоянии (256x256)
- `slime-moving.png` - Спрайт слайма в движении (256x256)
- `slime-eating.png` - Спрайт слайма при поедании (256x256)
- `slime-damaged.png` - Спрайт слайма при повреждении (256x256)

**Требования:**
- Формат: PNG с прозрачностью (RGBA)
- Размер: 256x256 пикселей (будет масштабироваться в клиенте)
- Слайм должен быть по центру спрайта

## Папка `warrior/`

Спец-спрайты для Warrior класса (большие колючки, агрессивный вид).

**Файлы:**
- `warrior-idle.png` - Воин в спокойном состоянии (256x256)
- `warrior-moving.png` - Воин в движении (256x256)
- `warrior-attacking.png` - Воин в атаке (256x256)
- `warrior-damaged.png` - Воин при повреждении (256x256)

**Характеристики:**
- Шипы/колючки по периметру
- Агрессивная поза
- Кровавые/боевые эффекты

## Папка `collector/`

Спец-спрайты для Collector класса (больший размер, спокойный вид).

**Файлы:**
- `collector-idle.png` - Сборщик в спокойном состоянии (256x256)
- `collector-moving.png` - Сборщик в движении (256x256)
- `collector-absorbing.png` - Сборщик при поглощении (256x256)
- `collector-damaged.png` - Сборщик при повреждении (256x256)

**Характеристики:**
- Более округлая форма
- Пассивный вид
- "Магические" эффекты поглощения

## Использование в клиенте

```typescript
// Загрузка спрайта
const spriteUrl = `/sprites/slimes/base/slime-idle.png`;
const img = new Image();
img.src = spriteUrl;

// Рендеринг на Canvas
ctx.drawImage(img, screenX - size/2, screenY - size/2, size, size);
```

## Варианты скинов (будущее)

В будущем можно добавить:
- `skins/` - Альтернативные скины (праздничные, классические, и т.д.)
- `accessories/` - Аксессуары (шляпы, очки, корона)
- `effects/` - Партикулы и эффекты (следы, взрывы)

## Цветовые коды игроков

Для динамического окрашивания спрайтов используются следующие правила:

- **Base класс** (классId=0): Зелёный (#9be070)
- **Warrior** (классId=1): Красный оттенок
- **Collector** (классId=2): Синий оттенок

Клиент может применять модуляцию цвета к спрайтам на основе классId игрока.
